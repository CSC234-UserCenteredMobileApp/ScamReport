## Change summary

> New \`downloadFile(bucket, path)\` Supabase helper returns raw bytes. Admin report PDF route fetches up to 5 evidence images (JPEG/PNG, ≤4 MB each), base64-encodes them, and passes an \`EvidenceImageMap\` to the template. \`reportTemplate\` renders each image inline below the metadata table with a muted filename + mime caption. PDF / non-image kinds still appear as text rows.
